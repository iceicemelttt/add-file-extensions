# add-file-extensions

Add some file extensions to a bunch of files.

## use

```js
var addExtension = require('add-file-extensions')

addExtensions('/path/to/folder', 'doc', function(err) {
	if (!err) {
		// all files in folder have `doc` extension added
	}
})
```
